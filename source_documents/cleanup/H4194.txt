VENSTRES LANDSSTYRE:
Venstre-utfordringer til verdikommisjonen
Venstres landsstyre vil peke på følgende utfordringer for kommisjonen som skal drøfte hvordan det kan skapes en bred verdimessig og samfunnsetisk mobilisering (verdikommisjonen).
 hvordan mennesker kan være borgere og ikke kunder og klienter.
 hvordan menneskeverdet kan vernes mot lønnsomhets- og nytteberegning.
 hvordan miljøhensyn på kort og lang sikt kan få gjennomslag for å verne og forvalte naturarven.
 hvordan diskrimisjon av det enkelte menneskets ukrenkelige verdi uansett kjønn, religion, legning og etnisk tilhørighet kan forhindres.
 hvordan familie, nærmiljø og skole kan gi barn og ungdom en trygg oppvekst.
 hvordan det kulturelle mangfold kan sikres som en grunnleggende verdi.
 hvordan en kan utvikle en global etikk som går lenger enn menneskerettighetene.
